Распознавание пломбы и контейнера

Форма шлёт multipart/form-data в поле files. Ответ API откроется в новой вкладке в виде JSON.

Эндпоинты: /RecognizeSealNumber и /RecognizeContainerNumber. Если откроешь этот файл локально, такой способ работает стабильнее, чем fetch, потому что не упирается в CORS.
